홈주요 일정, 과제, 투표와 모바일 새로고침⌄

일정 요약 확인

  • 홈 상단의 주요 일정에는 가까운 시험·학사·행사가, 과제 D-Day에는 가까운 과제가 표시됩니다.
  • 항목을 누르면 캘린더의 해당 월로 이동하고, 그 날짜에 등록된 일정 창이 열립니다.
  • 선택한 일정에 세부 내용이 있으면 자동으로 펼쳐집니다.
  • 전체 보기를 누르면 캘린더 화면으로 이동합니다.

화면 이동과 새로고침

  • PC에서는 위쪽 메뉴를, 모바일에서는 오른쪽 위의 메뉴 버튼을 눌러 다른 화면으로 이동합니다.
  • 모바일에서 화면의 맨 위를 아래로 당기면 곰 발바닥이 차오릅니다. 발바닥이 모두 채워진 뒤 손을 놓으면 최신 내용으로 새로고침됩니다.

진행 중인 투표와 상단 캐릭터

  • 진행 중인 투표가 있으면 홈 상단에 제목, 작성자와 항목 수가 표시됩니다. 누르면 도구 화면의 해당 투표가 열립니다.
  • 상단의 작은 곰 발바닥을 누르면 단웅이와 단비가 가운데로 모여 순서대로 여러 동작을 보여줍니다. 다음 동작 순서는 로그인한 계정별로 저장됩니다.
시간표공통 수업 확인과 내 시간표 제작⌄

수업 추가

  • 전체 학생에게 공통으로 등록된 수업과 본인이 추가한 수업이 한 시간표에 함께 표시됩니다.
  • 과목명을 입력하고 원하는 색상을 고른 뒤 과목명 옆의 확인 버튼을 누릅니다.
  • 확인 버튼이 옅어지면 시간표에서 수업을 넣을 첫 시간부터 마지막 시간까지 세로로 드래그합니다.
  • 시간표는 평일 오전 8시부터 오후 8시까지 30분 단위로 선택할 수 있습니다.
  • 과목명을 수정하면 확인 상태가 해제되므로 다시 확인을 눌러야 합니다.
  • 본인이 추가한 수업은 자신의 시간표에만 저장됩니다.

확인과 삭제

  • 공통 수업과 개인 수업은 같은 시간표에 함께 표시되며 색상과 테두리 모양으로 구분됩니다.
  • 본인이 등록한 수업 블록의 × 버튼을 누르면 해당 수업을 삭제할 수 있습니다.
  • 모바일에서는 시간표를 좌우로 밀어 월요일부터 금요일까지 확인합니다.
공지사항공지 열람, 검색과 페이지 이동⌄

공지 확인

  • 공지 제목을 누르면 세부 내용이 펼쳐지고, 다시 누르면 접힙니다.
  • 검색창에 제목, 세부 내용 또는 분류를 입력해 필요한 공지를 찾을 수 있습니다.
  • 검색창의 × 버튼을 누르면 검색어가 지워지고 전체 공지 목록으로 돌아갑니다.
  • 공지는 한 페이지에 10개씩 표시되며 아래쪽의 페이지 번호로 이전 공지를 확인합니다.
  • 고정된 공지는 일반 공지보다 위에 표시됩니다.
  • 세부 내용은 길이 제한 없이 확인할 수 있으며, 첨부 사진은 한 게시물에 최대 5장까지 표시됩니다.
  • 첨부 사진을 누르면 큰 화면으로 확인할 수 있습니다. PDF·Word·HWP 같은 문서 파일 첨부는 지원하지 않습니다.
캘린더월별 달력, 분류별 목록, 개인 일정과 D-Day⌄

달력과 검색

  • 날짜를 누르면 그날의 모든 일정이 열립니다. 세부 내용이 있는 항목은 제목을 눌러 펼치거나 접을 수 있습니다.
  • 달력 위의 화살표로 이전·다음 달로 이동하고 오늘을 누르면 현재 달로 돌아옵니다.
  • 검색창에서는 일정명, 세부 내용과 분류를 검색할 수 있습니다. 결과를 누르면 해당 날짜가 열리고 선택한 세부 내용이 자동으로 펼쳐집니다.
  • 주요 일정과 과제 D-Day를 눌러도 같은 방식으로 해당 날짜를 확인할 수 있습니다.

분류별 일정 모아보기

  • 달력 아래의 학사·시험·행사·할 일·과제·모임·방학·공휴일 버튼에서 분류별 일정을 모아볼 수 있습니다.
  • 분류 목록은 처음에는 접혀 있습니다. 버튼의 숫자는 오늘부터 1개월 안에 등록된 해당 분류의 일정 개수입니다.
  • 버튼을 펼치면 오늘부터 한 달 뒤 같은 날짜까지의 일정이 날짜순으로 표시됩니다. 해당 기간과 겹치는 기간 일정도 포함됩니다.
  • 목록의 일정을 누르면 달력에서 해당 날짜가 열리고 세부 내용도 자동으로 펼쳐집니다. 개인 일정에는 개인 표시가 붙습니다.

일정 추가

  • 날짜를 직접 누른 뒤 시작일·종료일, 제목, 분류와 세부 내용을 입력해 일정을 저장합니다.
  • 기간 일정은 시작일부터 종료일까지 달력의 각 날짜에 표시됩니다.
  • 일반 사용자가 추가한 일정은 본인에게만 보이고, 연필·휴지통 버튼으로 수정하거나 삭제할 수 있습니다.
  • 개인 일정에는 세부 내용을 길게 입력하고 사진을 최대 5장까지 첨부할 수 있습니다. PDF·Word·HWP 같은 문서 파일은 지원하지 않습니다.
  • 저장한 개인 일정은 다른 사용자에게 보이지 않으며 본인 계정으로 로그인했을 때만 표시됩니다.
알림새 알림 확인과 삭제⌄

알림 확인

  • 상단 종 모양에 표시된 숫자는 아직 확인하지 않은 알림 개수입니다.
  • 알림 제목을 누르면 최신 세부 내용이 펼쳐지고 확인한 알림은 옅게 표시됩니다.
  • 같은 알림이 다시 발송되더라도 확인 전에는 목록에 중복으로 쌓이지 않습니다.
  • 이미 확인한 알림이 다시 발송되면 최신 내용으로 목록 맨 위에 다시 표시됩니다.
  • 제출 확인이 필요한 항목에서 본인이 미제출로 표시되면 개인 알림을 받을 수 있습니다. 제출 처리가 완료되면 관련 알림도 자동으로 확인 상태가 됩니다.
  • 알림은 한 페이지에 10개씩 표시되며 페이지 번호로 이전 알림을 확인할 수 있습니다.

알림 정리

  • 개별 삭제와 모두 삭제를 이용해 본인의 알림 목록을 정리할 수 있습니다.
  • 원본 공지나 일정이 삭제되면 관련 알림도 함께 사라집니다.
자료실Google Drive 자료와 사진 공유⌄

공유 자료 확인

  • 자료실 위쪽의 Google Drive에서 열기를 누르면 26학번 공동 자료실이 새 창에서 열립니다.
  • Google Form으로 제출한 이메일의 Google 계정으로 로그인하면 공유 자료를 확인할 수 있습니다.

사진 올리기

  • 사진 갤러리 위의 업로드 영역에서 사진 파일을 선택하고 해시태그를 입력한 뒤 사진 올리기를 누릅니다.
  • 해시태그는 공백이나 쉼표로 구분하며 # 기호를 쓰지 않아도 자동으로 붙습니다.
  • 별도의 사진 설명 입력란은 없으며, 필요한 설명은 해시태그로 간단히 남깁니다.
  • 업로드가 끝날 때까지 페이지를 닫거나 새로고침하지 마세요.

사진 확인과 삭제

  • 사진 카드에는 해시태그와 사진을 올린 사람의 실명이 표시됩니다.
  • 사진을 누르면 큰 화면으로 볼 수 있습니다.
  • 자신이 올린 사진에 표시되는 삭제 버튼으로 해당 사진을 삭제할 수 있습니다.
도구룰렛, 사다리타기, 팀 나누기와 투표⌄

도구 열기와 명단 선택

  • 사용할 항목의 열기를 누르면 해당 도구가 펼쳐지고 다른 도구는 접힙니다.
  • 전체 룰렛은 등록된 동기 중 한 명을 무작위로 뽑고, 직접 입력 룰렛은 입력한 이름을 나누어 원판을 만듭니다.
  • 이름 원판·사다리타기·팀 나누기에서는 동기 명단에서 선택을 펼쳐 이름 버튼으로 참가자를 입력할 수 있습니다.
  • 이름 원판에서는 같은 사람을 여러 번 추가할 수 있고, − 버튼으로 한 칸씩 뺄 수 있습니다. 사다리타기와 팀 나누기에서는 한 사람을 한 번만 선택할 수 있습니다.
  • 도구 바깥의 명단 초기화를 누르면 해당 항목에 입력된 참가자 이름이 모두 지워집니다.

사다리타기

  • 참가 인원을 정한 뒤 참가자와 도착 결과를 입력하고 사다리 만들기를 누릅니다. 참가자와 결과 칸은 처음에는 비어 있습니다.
  • 개별 결과를 선택해 확인하거나 결과 표로 보기를 눌러 전체 연결 결과를 볼 수 있습니다.
  • 모바일에서 사다리가 넓으면 게임 영역을 좌우로 밀어 확인합니다.

팀 나누기

  • 참가 인원과 팀 개수를 정한 뒤 직접 입력하거나 동기 명단에서 참가자를 선택하고 팀 나누기를 누릅니다.
  • 각 팀의 인원 차이가 1명 이하가 되도록 자동으로 균등하게 배정됩니다.
  • 서로 다른 팀에 들어가야 하는 사람은 위에서 선택한 참가자 명단의 버튼으로 지정할 수 있습니다. 지정 인원은 팀 개수 이하로 제한됩니다.

투표

  • 누구나 제목과 항목을 입력해 투표를 만들 수 있으며 작성자의 이름이 함께 표시됩니다.
  • 투표 생성 전에 투표 후 수정 허용, 종료 전 실시간 결과 공개, 참여자의 항목 추가 허용 여부를 선택합니다.
  • 투표 후 수정이 허용된 경우 종료 전까지 선택을 바꿀 수 있습니다. 실시간 결과 공개가 꺼져 있으면 종료 전에는 다른 사람의 결과가 보이지 않습니다.
  • 투표 작성자는 투표 내용 수정·삭제, 참여자·미참여자 확인과 투표 종료를 할 수 있습니다.
  • 투표를 종료하면 설정과 관계없이 모든 사용자에게 최종 결과가 공개됩니다.
미니게임천호지 달리기와 산책⌄

게임 설정과 시작

  • 상단 메뉴에서 미니게임을 누르면 게임 전용 화면으로 이동합니다.
  • 불 끄기·불 켜기로 낮과 밤을 바꾸고, 수달 또는 거북이를 선택합니다. 선택한 설정은 현재 기기에 저장됩니다.
  • 달리기는 장애물을 피하며 기록에 도전하는 모드이고, 산책은 장애물 없이 풍경과 바퀴 수를 확인하는 모드입니다.
  • 전체화면을 누르면 배경과 캐릭터에 집중해 플레이할 수 있습니다. 게임 화면 안의 × 버튼으로 종료합니다.
  • 게임 화면 위의 플레이 방법을 펼치면 현재 기기에 맞는 조작 방법을 다시 확인할 수 있습니다.

달리기 조작

  • 게임 화면을 누르면 달리기가 시작되며, 다시 누르면 점프합니다. 공중에서 한 번 더 누르면 이중 점프합니다.
  • 모바일에서는 왼쪽 아래의 파란 삼각형 버튼으로 이동합니다. PC에서는 ←·→ 또는 A·D 키를 사용합니다.
  • 점프 중에는 오른쪽 위의 아래 삼각형이 활성화됩니다. 이를 누르면 빠르게 착지합니다. PC에서는 ↓ 또는 S 키를 사용할 수 있습니다.
  • 60종의 장애물이 등장하며, 낮은 장애물은 한 개 또는 두세 개가 거의 붙은 묶음으로 나타날 수 있습니다.
  • 백로는 높은 순찰 비행 뒤 수평 비행 또는 V자 급강하를 시도합니다. 수평 비행은 캐릭터가 있는 쪽의 반대편에서 접근하며, 급강하는 판단 순간의 캐릭터 위치가 붉게 표시됩니다. 산책 모드에서는 임의의 방향으로 천천히 활공하고 가끔 오른쪽 아래 산책로에 내려앉습니다.
  • 장애물에 충돌하면 달리기가 종료되고, 시간이 지날수록 이동 속도와 난이도가 높아집니다.

기록과 랭킹

  • 최고 기록은 수달 모바일·수달 PC·거북이 모바일·거북이 PC 네 종류로 구분되어 저장됩니다.
  • 터치 중심 기기는 모바일, 키보드·마우스 중심 기기는 PC 기록으로 자동 분류됩니다.
  • 기록 영역 왼쪽에서 수달·거북이, 오른쪽에서 PC·모바일을 선택해 각 분류의 내 최고 기록, 내 순위와 전체 최고 기록을 확인합니다.
  • 내 순위와 전체 최고 기록은 일반 구성원 기록을 기준으로 계산됩니다.
  • 룰렛·사다리타기·팀 나누기·투표는 기존 도구 화면에서 계속 이용할 수 있습니다.

화면과 로그인 설정

  • 배너 색상에서 원하는 색을 선택하면 홈과 각 화면 상단에 적용되며 현재 기기에 저장됩니다.
  • 기본 색상을 누르면 사이트 기본 배너 색상으로 돌아갑니다.
  • 단웅이 단비 표시를 끄면 위쪽의 3D 캐릭터 애니메이션이 숨겨지고, 다시 켜면 즉시 표시됩니다. 선택한 상태는 현재 기기에 저장됩니다.
  • 로그인 유지를 켜면 브라우저를 닫았다가 다시 열어도 이 기기에서 로그인 상태가 유지됩니다.
  • 공용 기기에서는 로그인 유지를 끄고 사용 후 반드시 로그아웃하세요.

비밀번호 변경과 문제 해결

  • Google Form 제출 확인이 표시되는 경우, 양식을 제출한 뒤 완료 체크박스를 선택해야 비밀번호 변경 버튼이 활성화됩니다.
  • 현재 비밀번호를 입력하고 8자 이상의 새 비밀번호를 두 번 입력한 뒤 비밀번호 변경을 누릅니다.
  • 새 비밀번호로 공용 초기 비밀번호를 다시 사용할 수 없습니다.
  • 비밀번호 표시를 누르면 새 비밀번호 입력 내용을 확인할 수 있고, 비밀번호 숨기기를 누르면 다시 가려집니다.
  • 변경 성공 메시지가 표시된 뒤부터 새 비밀번호로 로그인합니다.
  • 비밀번호를 잊어 로그인할 수 없다면 홈페이지 담당자에게 재설정을 요청하세요.
  • 로그아웃을 누르면 현재 기기의 계정 연결이 종료됩니다.
  • 화면이 갱신되지 않으면 PC에서 Ctrl + F5를 누르거나 모바일 브라우저를 새로고침하세요.
  • 계속 문제가 있으면 로그아웃한 뒤 다시 로그인해 보세요.

갤럭시에서 설치

  • Chrome에서 홈페이지를 연 뒤 오른쪽 위의 ⋮ 메뉴를 누릅니다.
  • 설치 및 바로가기 만들기를 선택합니다.
  • 설치 버튼이 나타나면 누릅니다. 설치가 완료되면 홈 화면이나 앱 목록에서 의과대학 26학번 앱을 확인할 수 있습니다.

아이폰에서 설치

  • Chrome에서 홈페이지를 연 뒤 오른쪽 아래의 공유 버튼을 누릅니다.
  • 공유 메뉴를 위로 올려 홈 화면에 추가를 선택하고, 오른쪽 위의 추가를 누릅니다.
  • 추가가 완료되면 홈 화면이나 앱 보관함에서 의과대학 26학번 아이콘을 확인하고 앱처럼 실행할 수 있습니다.

설치 후 이용

  • 설치 후에도 내용과 기능은 웹사이트와 동일하며, 인터넷에 연결되어 있을 때 최신 정보가 반영됩니다.
  • 업데이트가 바로 보이지 않으면 설치한 앱을 완전히 종료한 뒤 다시 실행해 주세요.
