로그인과 메뉴 이동

  • 이름과 비밀번호로 로그인하면 홈페이지의 모든 메뉴를 이용할 수 있습니다.
  • PC에서는 상단 메뉴를, 모바일에서는 오른쪽 위의 메뉴 버튼을 눌러 페이지를 이동합니다.
  • 상단의 종 모양 버튼에 표시된 숫자는 아직 확인하지 않은 알림 개수입니다.

공지사항과 알림

  • 공지 제목을 누르면 접혀 있던 세부 내용이 열립니다. 다시 누르면 접힙니다.
  • 검색창에서 공지 제목, 세부 내용 또는 분류를 검색할 수 있습니다.
  • 공지는 한 페이지에 10개씩 표시되며, 아래쪽 페이지 번호로 이전 공지를 확인합니다.
  • 알림을 누르면 해당 공지나 일정의 최신 세부 내용을 바로 확인할 수 있습니다.
  • 확인한 알림은 옅게 표시되며, 개별 삭제 또는 모두 삭제를 사용할 수 있습니다.

학사일정과 D-Day

  • 날짜를 누르면 그날의 일정과 세부 내용이 열립니다.
  • 상단 화살표로 이전·다음 달로 이동하고, 오늘 버튼으로 현재 달로 돌아옵니다.
  • 검색창에서 일정명, 세부 내용 또는 일정 분류를 검색할 수 있습니다.
  • 여러 날 동안 이어지는 일정은 시작일부터 종료일까지 각 날짜에 표시됩니다.
  • 가까운 시험·학사·행사는 주요 일정 D-Day에, 과제로 분류된 일정은 과제 D-Day에 표시됩니다.
  • D-Day 카드를 누르면 해당 월과 날짜가 자동으로 열립니다.

시간표

  • 학번 정보 페이지에서 과목명과 색상을 선택한 뒤 원하는 요일의 시간대를 세로로 드래그합니다.
  • 시간표는 오전 8시부터 오후 8시까지 30분 단위로 설정할 수 있습니다.
  • 관리자는 모두에게 보이는 전체 공통 수업과 자신의 개인 수업을 선택해 추가할 수 있습니다.
  • 일반 사용자가 추가한 수업은 본인의 개인 시간표에만 저장됩니다.
  • 수업 블록 오른쪽 위의 × 버튼을 누르면 해당 수업을 삭제할 수 있습니다.

갤러리

  • 사진 파일을 선택하고 해시태그를 입력한 뒤 사진 올리기를 누릅니다.
  • 해시태그는 공백이나 쉼표로 구분하며, # 기호는 자동으로 붙습니다.
  • 사진 아래에는 올린 사람의 이름이 표시됩니다.
  • 사진을 누르면 크게 볼 수 있고, 자신이 올린 사진은 삭제할 수 있습니다.

미니게임과 투표

  • 각 게임의 게임 열기 버튼을 누르면 해당 게임만 펼쳐집니다.
  • 룰렛은 동기 전체 또는 직접 입력한 이름 중에서 무작위로 한 명을 뽑습니다.
  • 사다리타기는 참가자와 도착 결과를 입력한 후 결과를 개별 또는 표로 확인합니다.
  • 투표는 누구나 만들 수 있으며, 작성자의 이름이 함께 표시됩니다.
  • 투표를 만들 때 선택 수정, 실시간 결과 공개, 참여자의 항목 추가 허용 여부를 정할 수 있습니다.
  • 투표가 종료되면 모든 참여자에게 최종 결과가 표시됩니다.

설정과 계정 관리

  • 배너 색상을 선택하면 홈 화면과 각 페이지의 상단 배너 색상이 바로 변경됩니다.
  • 자동 로그인을 켜면 브라우저를 다시 열어도 로그인 상태가 유지됩니다.
  • 비밀번호 변경에는 현재 비밀번호와 6자 이상의 새 비밀번호가 필요합니다.
  • 공용 기기에서는 자동 로그인을 끄고 사용이 끝난 뒤 로그아웃하세요.

화면이 제대로 갱신되지 않을 때

  • PC에서는 Ctrl + F5로 완전히 새로고침합니다.
  • 모바일에서는 페이지를 닫았다가 다시 열거나 브라우저를 새로고침합니다.
  • 로그인이 풀렸다면 로그인 페이지에서 다시 로그인하세요.
