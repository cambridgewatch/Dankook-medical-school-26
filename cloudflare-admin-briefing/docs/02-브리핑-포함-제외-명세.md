# 브리핑 포함·제외 데이터 명세

## 포함

### 전체 일정
- 오늘부터 최대 7일 범위와 겹치는 일정
- 여러 날 일정은 시작일과 종료일이 범위에 겹치면 포함
- 제목, 세부 내용, 분류, 시작일, 종료일, 첨부 개수
- `migrationMarker`와 `hidden: true` 문서는 제외

### 관리자 개인 일정
- Secret `ADMIN_UID` 경로의 일정만 조회
- 외부 응답에는 UID, 작성자 이름, 작성자 이메일을 넣지 않음

### 공지
- 고정 공지
- 최근 7일에 작성된 공지
- 제목, 내용, 분류, 고정 여부, 첨부 개수, 생성·수정 시각

### 알림
- 오늘 기준 최근 2일 또는 향후 7일 범위의 알림
- 제목, 내용, 유형, 연결 날짜, 생성·수정 시각
- `noticeId`, `calendarEventKey`는 내부 연결용이므로 외부 미반환

### 투표
- 진행 중인 투표
- 최근 7일 안에 종료된 투표
- 제목, 설명, 선택지 문구, 설정, 참여 인원
- 참여 인원은 `votes` 하위 컬렉션 문서 수만 사용

## 제외

- 학생 실명
- 회원 이메일
- Firebase UID
- 비밀번호 및 비밀번호 변경 기록
- 관리자 이름·이메일·UID
- 서비스 계정 JSON
- Action 인증키
- 사진·첨부파일 URL과 실제 파일 내용
- 투표자 이름
- 개별 투표 선택
- 선택지별 득표수
- 사용자 승인 상태
- 명단 전체

## 실명 제거 방식

- `members`의 `name`은 Worker 내부 치환 목록 생성에만 사용합니다.
- 제목·세부 내용·선택지에 등록된 이름이 있으면 `[이름]`으로 바꾼 뒤 반환합니다.
- 이메일 주소는 `[이메일]`로 바꿉니다.
- 명단에 없는 별칭이나 추가 이름은 `REDACT_TERMS` Secret으로 보완합니다.
