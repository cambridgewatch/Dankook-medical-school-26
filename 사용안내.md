# 단국대학교 의과대학 26학번 홈페이지 — 사용 안내

## 1. 열어보기
`index.html` 파일을 **더블클릭**하면 브라우저(크롬 등)에서 바로 열립니다.
설치할 것은 없습니다.

## 2. 폴더 구성
```
홈페이지/
├─ index.html       ← 홈(메인)
├─ about.html       ← 학번 소개 · 인사말
├─ notices.html     ← 공지사항 게시판
├─ calendar.html    ← 학사일정
├─ gallery.html     ← 사진 갤러리
├─ members.html     ← 동기 명단
├─ 사용안내.md       ← 이 파일
└─ assets/
   ├─ css/style.css ← 색상·디자인 (단국 네이비 컬러)
   ├─ js/main.js    ← 메뉴·검색·갤러리 동작
   └─ img/          ← 사진 (photo1~4)
```

## 3. 내용 수정하는 법
HTML 파일을 **메모장**이나 **VS Code**로 열어 글자만 바꾸면 됩니다.

- **인사말** → `about.html` 의 `<blockquote>` 안 문장 수정
- **공지 추가** → `notices.html` 의 `notice-item` 블록 하나를 복사해 붙여넣고 내용·날짜 변경
- **학사일정** → `calendar.html` 의 `tl-item` 블록 수정
- **사진 추가** → 사진을 `assets/img/` 에 넣고, `gallery.html` 의 `figure` 블록 복사 후 파일명만 변경
- **동기 명단** → `members.html` 의 `member` 블록 복사·수정 (이름과 `data-name` 값을 똑같이 맞추면 검색됨)

## 4. 색상 바꾸기
`assets/css/style.css` 맨 위 `:root` 부분의 색상 코드만 바꾸면 전체 색이 한 번에 바뀝니다.
- `--navy` : 메인 네이비
- `--blue` : 포인트 블루
- `--gold` : 포인트 골드

## 5. 인터넷에 무료로 올리기 (선택)
이 폴더를 그대로 **GitHub Pages**, **Netlify**, **Vercel** 같은 무료 호스팅에 올리면
전 세계 어디서나 주소로 접속할 수 있습니다. 필요하면 도와드릴게요.
